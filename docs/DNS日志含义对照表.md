# DNS中继服务器 - 日志标识含义对照表

| 日志标识 | 含义 | 出现场景 |
|---------|------|---------|
| **QUERY** | 收到客户端发来的 DNS 查询 | 每次有 nslookup 或浏览器请求时，打印查的是什么域名和类型 |
| **HIT** | 本地表命中 | 域名在 dnsrelay.txt 里配了普通 IP，直接返回 |
| **BLOCK** | 拦截命中 | 域名配了 0.0.0.0，返回"域名不存在"（NXDOMAIN） |
| **RELAY** | 转发上游 | 本地表没有该域名，转发给上游 DNS 去查询 |
| **REPLY** | 收到上游响应并返回给客户端 | 上游 DNS 回了结果，转发回客户端 |

## 一句话总结

- **QUERY** = 收到
- **HIT** = 本地有，直接回
- **BLOCK** = 黑名单，拦了
- **RELAY** = 本地没有，转发出去了
- **REPLY** = 转发的结果回来了，给客户端
